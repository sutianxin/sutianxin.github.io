Jetbrains全家桶破解

发表于2021-03-13|更新于2021-03-13|软件使用
|字数总计:339

资源下载，密码：d2l7

在破解完后，请不要随意移动插件的位置。

资源下载

1、下载软件安装包

访问 Jetbrains 的 官方网站 ，在导航栏 Developer Tool 选项中选择自己要下载的开发软件，这里以Pycharm为例。

点击 DOWNLOAD 选项进行下载

选择 Professional 进行下载

2、安装软件

双击第一步中下载的安装包

勾选 64-bit launcher，点击next

点击 Install 进行安装

安装完成后，点击 Finish 退出安装程序，运行 Pycharm ，进入试用界面

如果没有出现以上的界面，那么可以在上面给出的资源中解压 reset_script.zip 压缩包，并在 reset_script 目录下运行重置脚本，其中linux和mac运行 .sh 脚本，windows运行 .vbs 脚本。

进入 Pycharm 的主界面

3、破解

在主页面左上角选择 File -> Settings进入设置页面，然后选择 Plugins【插件】一栏

选择下载资源中的 BetterIntellJ 插件进行安装

点击 Apply 并重启软件

看到以下提示后重启即可。

用记事本打开资源中的 【激活补丁key】，复制全部内容，在 pycharm 主页面中选择【Help】-> 【Register】

点击 【Add New License】添加激活码

此时提示激活成功

DataGrip和IDEA的破解步骤与Pycharm大同小异，这里不再赘述。
